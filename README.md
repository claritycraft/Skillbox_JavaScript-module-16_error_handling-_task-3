### En:
#### Task 3
 What You Need to Do
You are given a simple function for testing error types.
There is a problem in the function: in both cases, it logs the message “Local error detected” to the console.
You need to fix the testErrorScope function so that in the second case, it logs the message “Global error detected” instead.

________________________________________________________________________________________________

### Ru:
#### Задача 3
Перед вами простая функция тестирования типа ошибок. В ней есть проблема. Функция в обоих случаях отображает сообщение в консоли «Обнаружена локальная ошибка». Необходимо исправить функцию testErrorScope так, чтобы во втором случае отображалось сообщение «Обнаружена глобальная ошибка».
